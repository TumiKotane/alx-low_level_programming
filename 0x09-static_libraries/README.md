C - Static libraries

[0. A library is not a luxury but one of the necessities of life](libmy.a)
To complete this task, you need to create a static library named "libmy.a" that includes all of the listed functions. If you have not implemented all of the functions, make sure to create empty functions with the correct prototypes. Additionally, ensure that the main.h header file includes prototypes for all of the functions included in the library. Finally, remember to push the main.h header file and the static library to your repository.

[1. Without libraries what have we? We have no past and no future](create_static_lib.sh)

Create a script named "create_static_lib.sh" that generates a static library called "liball.a". The script should include all the .c files present in the current directory as part of the library.
